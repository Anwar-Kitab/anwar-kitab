ANWAR KITAB WEBSITE

FILES
- index.html: website content
- styles.css: colors, layout, and mobile design
- script.js: products, filters, cart, and order-request features

CUSTOMIZE
1. Open script.js and edit the products list.
2. In index.html and script.js, replace:
   orders@anwarkitab.com
   @anwarkitab
   store hours
3. Add product photos later by replacing the emoji area with image tags.

PUBLISH FREE WITH NETLIFY
1. Go to https://app.netlify.com/drop
2. Drag the entire "anwar-kitab-website" folder onto the page.
3. Netlify will publish it and give you a free website address.

PUBLISH FREE WITH GITHUB PAGES
1. Create a free GitHub account.
2. Create a new public repository.
3. Upload index.html, styles.css, and script.js.
4. Open Settings > Pages.
5. Select the main branch and root folder, then save.

PAYMENTS
This starter website creates order requests through email. To accept card payments directly, connect a service such as Square, Stripe Payment Links, PayPal, Shopify Starter, or Ecwid.
